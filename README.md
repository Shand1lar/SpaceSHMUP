# spaceSHMUP
 
## Overview
 
This update focuses on audio enhancements, a new kill counter system, and weapon power-up functionality.
 
---
 
## Audio Improvements
 
### Enemy_4 Ship – Spawn Sound
Added a sound effect that plays when the `enemy_4` ship enters the screen. This gives the player an audio cue to alert them that a tougher enemy has appeared.
 
### Enemy_4 Ship – Destruction Sound
Added a distinct destruction sound effect that triggers when the `enemy_4` ship is destroyed.
 
### Hero Ship – Destruction Sound
Added a sound effect for when the hero ship is destroyed.

### Hero Ship - Shooting Sound
Added a sound effect for when the hero ship is shooting.
---
 
## Kill Counter
 
Added a kill counter to the game UI that tracks the total number of enemies destroyed during a run.
 
- Increments each time an enemy is destroyed
- Displayed on-screen so players can monitor it in real time
 
---
 
## Weapon Power-Ups
 
Implemented the weapon power-up system, allowing the hero ship to upgrade its weapons upon collecting power-up items. Players can now collect power-ups to enhance their firepower as they progress through the game.
 
---
 
## Explosion

---

Created explosion effect for when `enemy_4` dies which pairs a bit better with the sound.
